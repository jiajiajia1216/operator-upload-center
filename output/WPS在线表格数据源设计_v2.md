# 操盘手排名看板系统 - WPS在线表格数据源设计

## 一、数据需求总览

当前看板系统需要 **3个核心数据表**，支持在线读取和更新：

| 序号 | 表格名称 | 更新频率 | 用途 | 主键 |
|------|----------|----------|------|------|
| 1 | 门店过程数据 | 每日 | 月度业绩看板 | 日期+门店名称 |
| 2 | 基建表 | 每月 | 门店映射关系 | 门店名称 |
| 3 | 排名数据 | 每月 | 操盘手效率指标 | 操盘手姓名 |

| 门店业绩同比值 | 数字 | 去年同期业绩金额 | 2028.5 |
| 门店业绩环比值 | 数字 | 上月同期业绩金额 | 7758.2 |
| 门店成交单量 | 数字 | 当日成交订单数 | 3 |
| 询价量 | 数字 | 当日询价次数 | 8 |
| 抖音本地回收额 | 数字 | 抖音渠道业绩 | 0 |
| 官网业绩 | 数字 | 官网渠道业绩 | 0 |
| BD业绩 | 数字 | BD渠道业绩 | 0 |

### 2.2 可选字段（扩展分析）
| 字段名 | 数据类型 | 说明 |
|--------|----------|------|

## 四、表格3：排名数据（每月更新）

### 4.1 操盘手维度字段
| 字段名 | 数据类型 | 说明 | 示例 |
|--------|----------|------|------|
| 操盘手姓名 | 文本 | 主键 | 王宏 |
| 区域 | 文本 | 所属区域 | 西部区域 |
| 城市 | 文本 | 所在城市 | 西宁市 |
| 商户 | 文本 | 主要商户 | ... |
| 门店数 | 数字 | 负责门店数量 | 7 |

### 4.2 效率指标字段（核心）
| 字段名 | 数据类型 | 说明 | 计算方式 |
|--------|----------|------|----------|
| 企微回复率 | 百分比 | 工作时间回复率 | ≥90%=10分 |
| 询价转化率 | 百分比 | 有效报价→成交 | ≥20%=15分 |
| 拉新店日均 | 数字 | 日均拉新数量 | ≥9=10分 |
| O2O效率 | 百分比 | O2O提交成交率 | ≥60%=15分 |
| 日均无人时长 | 数字 | 分钟为单位 | ≤80min=10分 |

### 4.3 绩效评估字段
| 字段名 | 数据类型 | 说明 | 计算方式 |
|--------|----------|------|----------|
| 巡店次数 | 数字 | 执行人=操盘手 | ≥门店数=10分 |
| 培训场次 | 数字 | 创建人=操盘手 | ≥2场=10分 |
| 培训覆盖率 | 百分比 | 培训覆盖比例 | ≥100%=满分 |
| 会议次数 | 数字 | 创建人=操盘手 | ≥11次=10分 |
| 通报次数 | 数字 | 创建人=操盘手 | ≥天数=10分 |

### 4.4 评分与排名字段
| 字段名 | 数据类型 | 说明 | 备注 |
|--------|----------|------|------|
| 总分 | 数字 | 各维度得分汇总 | 满分100 |
| 排名 | 数字 | 按总分排序 | 1,2,3... |
| 累计百分比 | 百分比 | 排名累计占比 | 用于ABCD分级 |
| 等级 | 文本 | A/B/C/D | A≤20%, B≤80%, C≤90%, D>90% |

## 五、数据关联关系

```
门店过程数据 (每日) ──门店名称──→ 基建表 (每月)
                    ↓
               区域/城市/操盘手
                    ↓
                排名数据 (每月)
```

## 六、WPS在线表格创建建议

### 方案A：单个表格，3个工作表
- 工作表1：门店过程数据
- 工作表2：基建表  
- 工作表3：排名数据

### 方案B：3个独立表格
- 表格1：加盟商过程数据_YYYYMM
- 表格2：加盟门店基建表_YYYYMM
- 表格3：操盘手排名数据_YYYYMM

**推荐方案A**，便于数据关联和维护。

## 七、数据更新流程

1. **每日更新**：上传CSV → 解析 → 追加到"门店过程数据"工作表
2. **每月更新**：上传新Excel → 替换"基建表"和"排名数据"工作表
3. **数据校验**：检查门店名称一致性，确保能正确关联
